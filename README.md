# React Forms

A learning project focused on building forms with React.

## Topics Covered

- Form validation
- State management
- Component composition

## Tech Stack

- React
- Vite
- ESLint

## Getting Started

```bash
npm install
```

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run lint` | Run ESLint |
| `npm run preview` | Preview production build |
